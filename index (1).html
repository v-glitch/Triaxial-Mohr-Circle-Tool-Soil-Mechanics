FORMULAS USED 
Notation:  C, C′ = circle centers (stress coordinates)
           c, c′ = cohesion (envelope intercepts) — different things!

σ₁ = σ₃ + qf                      total major principal stress
σ′ = σ − Δuf                      effective stress
C  = (σ₁ + σ₃) / 2                 center, total circle
C′ = (σ′₁ + σ′₃) / 2               center, effective circle
R  = (σ₁ − σ₃) / 2 = qf / 2       radius (same for both circles)

Envelope τ = c + σ·tanφ tangent to circle (center C, radius R):
    C·sinφ + c·cosφ = R
    φ = asin( R / √(C² + c²) ) − atan( c / C )
    special case c = 0:   sinφ′ = R / C′

Failure plane angle:   αf = 45° + φ′/2   (from major principal plane)
Failure point (tangency):   σ′ff = C′ − R·sinφ′ ,   τff = R·cosφ′
Pole construction:  pole of a triaxial circle is at (σ′₃, 0);
    a line from the pole at αf passes through the failure point
Borderline:  cmax = R = qf/2  →  φ = 0 (horizontal envelope, UU-style)
Built for educational use · 2026 · verify results independently · Single-specimen interpretation: c′ is user-specified, φ′ is the tangent solution. Multi-specimen envelope fitting is the proper way to determine c′ and φ′ together.
